# 🚖 RideX Smart Mobility Platform

RideX is a full-stack smart transportation and ride management platform designed to simulate real-world ride booking applications like Uber and Ola. The system supports secure authentication, ride booking, driver management, ride lifecycle tracking, OTP verification, analytics, and admin controls.

# ⚙️ How To Run

## 1️⃣ Clone Repository

```bash
git clone YOUR_REPOSITORY_LINK
```

---

## 2️⃣ Open Project

Open project in VS Code or IntelliJ.

---

## 3️⃣ Configure Database

Update:

```properties
spring.datasource.username=YOUR_USERNAME
spring.datasource.password=YOUR_PASSWORD
```

inside:

```text
application.properties
```

---

## 4️⃣ Run MySQL

Create database:

```sql
CREATE DATABASE ridex;
```

---

## 5️⃣ Run Spring Boot Application

```bash
mvn spring-boot:run
```

---

## 6️⃣ Open Frontend

Run using Live Server or browser.
